# Multi-Source-News-Aggregator
A powerful desktop application built with Tkinter that collects real-time news from multiple sources (RSS + API), analyzes sentiment, identifies trending topics, and delivers smart news insights with email digest support — all in one clean interface.


## Key Features
News Aggregation
Fetches real-time news from multiple RSS feeds (BBC, CNN, Reuters, etc.)
Optional integration with NewsAPI
Removes duplicate articles automatically

## Smart Search & Filtering
Search news by keyword
Filter by:
Category (Technology, Business, Sports, etc.)
Source (BBC, CNN, etc.)

## Sentiment Analysis
Detects article tone:
Positive
 Negative
 Neutral
Displays sentiment score for better insights

## Trending Topics
Extracts and displays top trending keywords
Helps identify what's hot in the news

##Email Digest System
Send top news articles via email
Supports Gmail & Outlook SMTP
HTML formatted professional email report

## Data Management
Save news as:
JSON
CSV
Load saved articles anytime

##Interactive GUI
Clean and modern UI using Tkinter
Pagination system for large datasets
Detailed article preview
Open articles directly in browser

##Technologies Used
Python (Core Logic)
Tkinter (GUI)
RSS/XML Parsing
SMTP (Email System)
JSON & CSV Handling
Multithreading


## How It Works
Fetches news from RSS feeds & APIs
Parses and cleans the data
Categorizes articles automatically
Performs sentiment analysis
Displays results in a GUI dashboard
Allows export or email sharing
